# DESIGN.md — Особиста бібліотека "Amber Shelf"

---

## 1. Концепція

> "Стара бібліотека при свічках" — цифровий простір, де кожна книга відчувається як фізичний об'єкт, а навігація — як прогулянка між стелажами.

---

## 2. Проблеми поточного дизайну

1. **Відсутній пошук** — при 35+ книгах неможливо швидко знайти конкретну назву чи автора без перегляду всіх карток
2. **Немає фільтру по difficulty** — теги `beginner/intermediate/advanced` існують у даних, але недоступні як окремий вимір фільтрації
3. **Обкладинка = заглушка** — велика літера на кольоровому фоні є placeholder-ом, не читабельним UI елементом
4. **Теги не інтерактивні** — `slavic-folklore`, `grimdark`, `heist` виглядають як декорація, хоча мають потенціал як фільтри
5. **Відсутнє сортування** — немає можливості впорядкувати за рейтингом, датою додавання, назвою A–Z
6. **Секційні лейбли малопомітні** — заголовки "Читаю зараз / Хочу прочитати / Прочитано" не виконують роль навігаційних орієнтирів
7. **Відсутній footer** — сторінка обривається без завершення, немає жодної контекстної інформації
8. **Дропдауни виглядають generic** — стандартні `<select>` елементи не відповідають "книжковому" характеру дизайну

---

## 3. Структура сторінки

```
┌─────────────────────────────────────────┐
│  HEADER                                 │
│  (лого + назва бібліотеки + лічильник)  │
├─────────────────────────────────────────┤
│  ПАНЕЛЬ ФІЛЬТРІВ                        │
│  (пошук + 4 фільтри + сортування)       │
├─────────────────────────────────────────┤
│  СЕКЦІЯ: "Читаю зараз"                  │
│  горизонтальний shelf-рядок             │
├─────────────────────────────────────────┤
│  СЕКЦІЯ: "Хочу прочитати"               │
│  grid карток                            │
├─────────────────────────────────────────┤
│  СЕКЦІЯ: "Прочитано"                    │
│  grid карток                            │
├─────────────────────────────────────────┤
│  FOOTER                                 │
│  (статистика + дата оновлення)          │
└─────────────────────────────────────────┘
```

### Header
Займає повну ширину. Ліворуч — іконка книги (unicode/SVG) + назва "Amber Shelf" великим serif шрифтом. Праворуч — лічильник "35 книг · 8 прочитано" у вигляді тихого тексту. Внизу header — тонка amber лінія-розділювач (1px).

### Панель фільтрів
Sticky-блок під header. Перший рядок: повноширинне поле пошуку. Другий рядок: 4 custom-дропдауни (Статус / Жанр / Difficulty / Рейтинг) + кнопка сортування праворуч. Фон панелі — на 2–3 відтінки темніше від фону сторінки, щоб виділятись як окрема зона.

### Секція "Читаю зараз"
Виділена секція з великим лейблом-заголовком і amber підкресленням. Книги відображаються горизонтальним рядком (scroll якщо > 4 книги). Це "featured" зона — картки тут трохи більші і мають підсвічений amber border.

### Секція "Хочу прочитати"
Стандартний grid. Заголовок секції — mid-розмір, нейтральний. Grid адаптивний: 2–5 колонок залежно від ширини вікна.

### Секція "Прочитано"
Аналогічно до "Хочу прочитати", але картки мають тонкий зелений акцент.

### Модальне вікно (деталі книги)
Overlay темніє фон. Модалка займає ~520px ширини. Ліворуч — "обкладинка" (стилізований блок з домінуючим кольором жанру + назва + автор великим шрифтом — імітація реальної обкладинки). Праворуч — мета-інформація: жанр, difficulty badge, рейтинг зірочками, теги як клікабельні чіпи, нотатки автора джерела (курсивом), власні нотатки.

### Footer
Фіксований низ сторінки. Три колонки: ліворуч — "Amber Shelf · особиста бібліотека", по центру — статистика (скільки прочитано, у процесі, в списку), праворуч — дата останнього оновлення.

## 6. Картка книги

### Розміри і форма
- Ширина: `160px` мінімум, до `200px`
- Висота: `260px`
- `border-radius: 3px 6px 6px 3px` — трохи більше заокруглення з правого боку (книжковий ефект)
- `box-shadow: 0 4px 16px rgba(0,0,0,0.5)` у спокійному стані

### Структура картки (зверху донизу)

**1. "Обкладинка" (72% висоти картки)**
- Фон: градієнт двох кольорів, унікальних для жанру
  - fantasy: `#1a1035` → `#3a1a6e` (темно-фіолетовий)
  - sci-fi: `#0a1a2a` → `#1a4060` (темно-синій)
  - horror: `#1a0808` → `#4a1010` (темно-бордо)
  - саморозвиток: `#0a1a10` → `#1a4028` (темно-зелений)
- По центру — назва книги у 3–4 рядки, шрифт Lora 13px, білий, text-align: center
- Внизу — ім'я автора Inter 10px, `rgba(255,255,255,0.6)`, text-align: center
- Зверху праворуч — маленький difficulty badge: "B" / "I" / "A" у кольоровому кружечку
  - B = `#4a7c59` зелений, I = `#c8922a` amber, A = `#8a3030` темно-червоний
- Spine-ефект: `border-left: 4px solid [status-color]`

**2. Нижній блок (28% висоти)**
- Фон: `#1c1814` (Surface)
- Padding: `8px 10px`
- Жанр: 1 тег-чіп (Inter 10px, amber border)
- Рейтинг: зірочки `★` Amber кольором (якщо є)
- Або: якщо немає рейтингу — просто жанр-бейдж

### Hover стан
- `box-shadow: 0 8px 28px rgba(200,146,42,0.2)` — amber glow
- `transform: translateY(-4px)`
- `transition: all 0.2s ease`

## 7. Фільтри і пошук

### Пошуковий рядок
- Розміщення: перший елемент в панелі фільтрів, повна ширина
- Висота: `42px`
- Фон: `#1c1814`
- Border: `1px solid #3a2e22`, при фокусі — `1px solid #c8922a`
- Placeholder: "Шукати за назвою або автором..." — Inter 13px, Text Muted
- Іконка пошуку ліворуч (SVG лупа, Amber колір)
- `border-radius: 6px`
- Фільтрація в реальному часі (debounce 200ms)

### Фільтри (4 нативних `<select>`, кастомно стилізованих)
- Статус: Всі / Читаю / Хочу прочитати / Прочитано
- Жанр: Всі / Fantasy / Sci-Fi / Саморозвиток / ...
- **Difficulty (новий):** Будь-яка / Beginner / Intermediate / Advanced
  - Логіка визначення: перевіряти наявність тегів `beginner`, `intermediate`, `advanced` у масиві tags книги
- Рейтинг: Будь-який / ★★★★★ / ★★★★+/ ★★★+

### Кнопка сортування
- Кнопка-пілюля праворуч: "A→Z ▾"
- Dropdown: Назва A–Z / Назва Z–A / Рейтинг ↓ / Дата додавання ↓

### Активні фільтри
- Якщо застосовано — під панеллю з'являється рядок чіпів: `[× Fantasy] [× Beginner]`
- Праворуч: кнопка "Скинути все"

## 9. Мобільна версія (< 768px)

### Header
- Назва залишається ліворуч, статистика ховається або показується як "35 книг"
- Padding: 14px 16px

### Панель фільтрів
- Пошуковий рядок — повна ширина
- Фільтри — горизонтальний scroll-рядок з компактними `<select>` (без міні-підписів)

### Grid карток
- 2 колонки фіксовані (`grid-template-columns: repeat(2, 1fr)`)
- Картки висота `220px`
- Текст назви на обкладинці — 11px

### Модальне вікно
- 100% ширини, max-height: 90vh
- `border-radius: 14px 14px 0 0` (bottom sheet стиль)
- Slide-up анімація

### Footer
- Три колонки → одна колонка, вертикально, text-align: center

## 10. Загальний тон

**Три прикметники:**

1. **Камерний** — відчуття особистого, приватного простору, ніби заходиш у когось у домашній кабінет з книжками, а не на публічний ресурс

2. **Теплий** — amber і коричневі тони, serif шрифт, м'яке освітлення карток створюють відчуття свічок і паперу, а не холодного екрану

3. **Зібраний** — кожна деталь на місці, немає зайвого шуму: фільтри компактні, картки лаконічні, інформація подається ієрархічно без перевантаження
